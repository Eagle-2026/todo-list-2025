A responsive **React** application to manage daily tasks.  
Users can **add, edit, delete, and mark tasks as completed**.  
Features include **task filtering**, **dark/light mode**, and **toast notifications**.

---

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks with confirmation
- Mark tasks as completed
- Filter tasks: All / Completed / Pending
- Dark/light mode toggle
- Toast notifications for actions
- Mobile responsive design
- Data persistence using localStorage

---

## Tech Stack
- React
- Material-UI (MUI)
- React Context API
- UUID for unique IDs
- LocalStorage for storing tasks
